## 前端开发规范

### 配色体系（B站风格）

采用接近B站风格的配色体系，视觉关键词：清爽、年轻、科技感、二次元友好但不过度装饰。

**品牌色：**

| 变量名              | 色值    | 用途                               |
| ------------------- | ------- | ---------------------------------- |
| `--bili-blue`       | #00A1D6 | 主色，用于按钮、链接、导航高亮     |
| `--bili-blue-hover` | #00B5E5 | 主色悬浮态                         |
| `--bili-pink`       | #FB7299 | 强调色，仅用于点赞、关注等用户反馈 |

**背景色：**

| 变量名            | 色值    | 用途         |
| ----------------- | ------- | ------------ |
| `--bg-white`      | #FFFFFF | 主背景       |
| `--bg-gray-light` | #FAFAFA | 页面背景     |
| `--bg-gray`       | #F4F4F4 | 次级背景     |
| `--bg-gray-dark`  | #E7E7E7 | 分隔线、网格 |

**文字色：**

| 变量名               | 色值    | 用途           |
| -------------------- | ------- | -------------- |
| `--text-primary`     | #18191C | 标题、重要文字 |
| `--text-regular`     | #61666D | 正文           |
| `--text-secondary`   | #9499A0 | 辅助信息       |
| `--text-placeholder` | #C9CCD0 | 占位符         |

**功能色：**

| 变量名            | 色值    | 用途     |
| ----------------- | ------- | -------- |
| `--color-success` | #00B578 | 成功提示 |
| `--color-error`   | #F56C6C | 错误提示 |
| `--color-warning` | #FF9736 | 警告提示 |

### 样式规范

1. **公共变量**：颜色变量定义在 `src/styles/common.css`，全局引入
2. **组件样式**：使用 `<style scoped>` 隔离，引用公共CSS变量
3. **选择器规范**：一个类选择器包含完整样式，避免原子类拆分
4. **过渡动画**：统一使用 0.1s（快速）或 0.2s（正常）
5. **圆角**：小组件 4px，中等 8px，卡片 12px
6. **避免使用**：渐变色、阴影、光晕、纯黑色、高饱和撞色
